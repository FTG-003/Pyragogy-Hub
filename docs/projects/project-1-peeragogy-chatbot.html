üìñAbout the Project

The Peeragogy ChatBot allows users to engage directly with the Peeragogy Handbook using natural language. Initially developed using Flowise and OpenAI APIs, the system enabled semantic chunking of the handbook's content and supported Q&A dialogue with embedded context. While the prototype was successfully completed, we are now entering a new phase of reimplementation, aiming for a more modular, open, and scalable AI-powered educational interface.
